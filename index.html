Cheat Sheet

  • 'flex-direction:'
    Sets the direction of the main axis, which can be row, column, row-reverse, or column-reverse. Reversing the direction flips the starting point on the main axis.
  • 'justify-content:'
    Determines how the items are aligned on the main axis. The available values are flex-start, space-around, space-between, and center.
  • 'align-items:'
    Specifies how the items are aligned on the cross axis. The options are baseline, stretch, flex-start, flex-end, and center. The cross axis is the opposite of the main axis.
  • 'flex-wrap:'
    Specifies whether the items should wrap when they exceed the container's width or height. The options are nowrap, wrap, and wrap-reverse.
  • 'align-content:'
    Determines how the items are aligned on the cross axis when there is extra space. The values include flex-start, flex-end, center, space-between, and space-around. This property has no effect on single line flex containers (i.e. ones with flex-wrap: nowrap).
  • 'flex-flow:'
    A shorthand property for flex-direction and flex-wrap. You can set both properties at once with this property.
  • 'align-self:'
    Sets the alignment of a single item on the cross axis. The values are flex-start, flex-end, center, stretch, and baseline.
  • 'flex-grow:'
    Determines how much an item should grow if there is extra space on the main axis. The value is a ratio.
  • 'flex-shrink:'
    Specifies how much an item should shrink if there is not enough space on the main axis. The value is also a ratio.
  • 'flex-basis:'
    Sets the initial size of an item before it enters the flexbox layout. This property is different from width because it applies to the height when the flex-direction is column.
  • 'flex:'
    A shorthand property for flex-grow, flex-shrink, and flex-basis. You can use it like this: flex: 1 0 auto; which sets flex-grow to 1, flex-shrink to 0, and flex-basis to auto.
  • 'order:'
    Specifies the order in which the items are displayed. The default value is 0, and you can use negative or positive values to change the order.
  • 'min-width' and 'min-height:'
    These properties can be used to set a minimum width or height for flex items. This is useful when you want to prevent an item from shrinking too much.
